    Convert XML Element To Array

    convert_xml_element_to_array(string $xml_element, int $recursion_depth) : array

    This is a utility function that takes a simpleXMLElement object and returns the contents as a nested associative PHP array, to make it easier to work with. It's called recursively to deal with an entire document. Adapted from http://www.ibm.com/developerworks/xml/library/x-xml2jsonphp/

    Parameters

    $xml_element

    string

    A single element, possibly containing children

    $recursion_depth

    int

    The depth of the call stack

    Returns

    arrayAll the tags in the XML element converted into an associative array

    Convert XML String To Array

    convert_xml_string_to_array(string $xml_string) : array

    Parameters

    $xml_string

    string

    The contents of an XML file

    Returns

    arrayAll the tags in the XML converted into a nested associative array

    Get First XML Attribute

    get_first_xml_attribute(string $tagname, string $attribute_name, array $index, array $vals, string $default) : string

    A utility function for a common case of XML parsing. It returns the value for the given attribute of the first occurrence of the given tag name. It requires the parsing arrays returned from xml_parse_into_struct()

    since Unknown

    Parameters

    $tagname

    string

    The name of the tag to search for

    $attribute_name

    string

    The name of the attribute within the tag to search for

    $index

    array

    The indices of all the tags parsed into an array by xml_parse_into_struct

    $vals

    array

    The values of all the tags parsed into an array by xml_parse_into_struct

    $default

    string

    If the tag isn't found, this value is returned instead. Defaults to an empty string

    Returns

    stringThe value of the first tag, or the default
