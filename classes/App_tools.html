    Replace Tags

    replace_tags(string $template, string $replace_tag, string $replace_value, bool $load_template) : string

    Given a template, replaces tags of the form {TAG_NAME} with ivars and config items.

    Stuff which gets replaced

    • {APP_NAME} with $this -> app_name
    • {APP_URL} with $this -> app_url
    • {APP_CLASS} with $this -> app_class
    • {SITE_NAME} with config_item('site_title')
    • {SITE_ADMIN} with config_item('site_admin_email')

    A single custom tag and value can be specified in $replace_tag and _value.

    todo Allow more custom tag flexibility by using an assoc. array of custom tags

    Parameters

    $template

    string

    The path to the template to process OR the string of the template

    $replace_tag

    string

    A single custom tagname (defaults to null)

    $replace_value

    string

    The value to replace instances of $replace_tag with

    $load_template

    bool

    Whether or not $template is a path to be loaded

    Returns

    stringThe data in $template with all tags substituted
