Add Activity

add_activity(array $activity_info, array $activity_data) : int | bool

Add a new activity into the database.

$activity_info should be an assoc. array where all the keys are activity table column names.

$activity_data is an array which will be json_encoded and stored in the mysql table

Example Parameters

$activity_info['site_id'] = 1;
$activity_info['user_id'] = 1;
$activity_info['verb'] = 'post';
$activity_info['module'] = 'home';
$activity_info['type'] = 'status';

$activity_data['title'] = false;
$activity_data['content'] = 'Hello world, this is a status update';
$activity_data['url'] = 'http://example.com/home/view/1';

Parameters

$activity_info

array

An array containing activity metadata

$activity_data

array

The activity data

Returns

intboolThe id of the inserted activity, or false on failure

Get Timeline

get_timeline(string | array $where, int $limit) : array

Gets an array of activity objects based on criteria in $where

$where can either be an array like array('column_name' => 'value to match') or a mysql WHERE clause, e.g. 'activity.site = 1'
